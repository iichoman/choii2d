# choii2d
## 게임 소개
2020년 9월에 출시한 spelunky2의 카피게임이다. spelunky2는 2d 횡스크롤 방식의 로그라이크 계열의 게임으로 스테이지를 진행하면서 몬스터를 피해 아이템을 파밍하며 다음스테이지로 통과하는 게임이다.
